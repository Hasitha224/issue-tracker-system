# Issue Tracker System
----------------------

Issue Tracker system that allows users to register, log in, and manage issues (CRUD Operations) with filtering and searching.

## Tech Stack
--------------------
### Frontend

* React (Vite)
* TypeScript
* Tailwind CSS
* React Query
* Axios

### Backend

* Node.js
* Express.js
* MongoDB (Atlas)
* Mongoose
* JWT Authentication

--------------------

## Project Structure

issue-tracker/
  |- issue-tracker-frontend/   # React application
  |- issue-tracker-backend/    # Express API

--------------------

## Live Demo

Frontend: https://issue-tracker-system-olive.vercel.app
Backend: https://issue-tracker-system-backend.onrender.com

--------------------
## Instructions

1. Clone the repository.
2. Setup Frontend and backend with the use of .env files.
3. User registration is needed to login to the system.
4. Able to manage issues after successful login.
--------------------
## Features

* User Registration & Login
* Create, Edit, Delete Issues
* Filter by Status, Severity, Priority
* Search Issues
* Pagination
* Mark issues as resolved
